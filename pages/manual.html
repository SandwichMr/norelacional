Proposito del sistema

Biblioteca Juvenil sirve para registrar y consultar autores, libros, usuarios y prestamos de una biblioteca escolar pequena.

Tecnologias utilizadas

El proyecto usa HTML, CSS, Bootstrap local, Bootstrap Icons local, JavaScript basico, SweetAlert2 local, Node.js, PHP y MongoDB.

Node se usa para correr el proyecto con el comando node server.js. PHP se usa separado para consultar y modificar MongoDB.

Como ejecutar el sistema

cd /home/leonardo/biblioteca_juvenil
node server.js

Despues se abre http://localhost:3000 en el navegador.

Base de datos y colecciones

La base se llama biblioteca_juvenil y usa las colecciones autores, libros, usuarios y prestamos.

  • autores: nombre, nacionalidad, correo
  • libros: titulo, genero, anio, autorId
  • usuarios: nombre, telefono, correo, grupo
  • prestamos: libroId, usuarioId, fechaPrestamo, fechaEntrega, estado

ObjectId y aggregate

Los campos autorId, libroId y usuarioId guardan ObjectId. En prestamos se usa aggregate con $lookup para unir libros y usuarios, $unwind para acomodar los datos y $project para mostrar solo los campos importantes.

CRUD paso a paso

Crear: llenar un formulario y guardar. Mostrar: entrar a la lista. Editar: usar el lapiz. Actualizar: guardar cambios. Eliminar: usar basura y confirmar con SweetAlert2.
